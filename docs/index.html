按鈕尚未指定 Streamlit 網址。請在下方展開說明完成部署，並編輯本檔的 STREAMLIT_APP_URL。

如何取得「線上儀表板」網址（Streamlit Community Cloud，免費）
  1. 到 share.streamlit.io 用 GitHub 帳號登入。
  2. New app → 選倉庫 yodev168/SteamAO_PD、Branch main、Main file phase5/app.py → Deploy。
  3. 部署成功後你會得到類似 https://＊＊＊.streamlit.app 的網址。複製到本專案 docs/index.html 裡的 STREAMLIT_APP_URL，再 push 一次，此按鈕就會一鍵開啟儀表板（含手機版）。
